# PiezoJet paper figure map

This map keeps each figure tied to one auditable question. Static figures are
regenerated by the named script and inspected in the compiled paper, not only
as standalone images.

| Paper role | Question answered | Figure | Evidence source | Status |
|---|---|---|---|---|
| Introduction | What are the maintained production, diagnostic, and isolated paths? | `piezojet_method_overview.pdf` | Maintained model and trainer execution paths | Current |
| Method | Why is one signed operator used across stable and unstable spectra? | `piezojet_operator_spectrum.pdf` | Analytic filter and first-order block system | Current |
| Method | Which full-Lambda directions are observable from available labels? | `vnext_identifiability_census.pdf` | Registered 4,939-record development-safe census | Current |
| Data | Where do structures and labels enter, and which branches receive gradients? | `piezojet_data_flow.pdf` | Canonical role map and registered cache accounting | Current |
| Data | How selection-biased is strict-Lambda completion? | `strict_completion_selection_bias.pdf` | v10 full-public label-coverage manifest | Current |
| Results | Does direct-U have same-ID capacity, and does that imply total-response advantage? | `direct_u_mechanism_boundary.pdf` | Registered samples32 capacity and three-seed val10 control | Current |
| Results | Was electronic failure a model-class floor, and which irrep dominated it? | `electronic_model_class_capacity.pdf` | Registered samples32 same-ID capacity summaries | Current |
| Results | How do A0, A1, and A1.5 generalize as N increases? | not generated | N=200 single-seed partial curve plus complete matched N=800/full Stage-3 summaries | N=200 may be tabulated as overfitting evidence; defer the comparison figure until matched N=800/full results exist |

The strict-completion and identifiability figures are deliberately separate:
completion is a source-label acceptance process, while identifiability is a
rank certificate for the information contained in the accepted observations.
